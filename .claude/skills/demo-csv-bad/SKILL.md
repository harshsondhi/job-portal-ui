# Demo CSV Bad

## Intent
The user wants to analyze and get a quick overview of a specific CSV file.

## Actions
1. Locate and read the CSV file specified by the user.
2. Extract and display the names of all columns in the file.
3. Calculate and report the total number of rows in the dataset.
4. Preview and summarize the data contained within the first 3 rows.
